meteorCF_BA
===========

Run application with: meteor --release 0.8.2

Required packages:
- Meteor (aquire via mrt add <package_name>):
  - reactive-table
  - npm
  - just-i18n
  - iron-router (v0.7.1)
    - iron-layout
    - iron-dynamic-template
    - iron-core
  - datatables
  - bootstrap-3
  - bootboxjs
  - blaze-layout
  - Kadira

- Npm (aquire via npm install <package_name>):
  - execSync
